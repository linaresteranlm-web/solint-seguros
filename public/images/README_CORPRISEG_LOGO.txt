Para que los reportes usen el logo real de CORPRISEG:

1. Copia el logo aquí:
   public/images/corpriseg-logo.png

2. Recomendado:
   - PNG
   - fondo transparente
   - ancho aprox. 600 px
   - alto aprox. 220 px

3. Si no colocas el logo, el PDF usará texto corporativo CORPRISEG.

4. Después de copiar el logo:
   CTRL + C
   rmdir /s /q .next
   npm run build
   npm run dev
